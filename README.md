# traduora-cli
## A package for talking to the Traduora API via npm scripts

### Install

yarn:
`yarn add -D PACKAGE_NAME`

npm:
`npm i --save-dev PACKAGE_NAME`

### Usage

Add the following commands to your `package.json` under the `scripts` section.

`tradoura addTerm`

`tradoura getStats`
